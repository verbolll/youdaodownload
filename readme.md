# 使用方法

1. 克隆仓库
`git clone https://github.com/verbolll/youdaodownload.git`
2. （可选）创建虚拟环境
3. 安装依赖包
`pip install -r requirements.txt`
4. 将要翻译的矢量pdf复制到本目录下
5. 更改`main.py`第23行文件名称
6. 运行`main.py`
